Project Description

Project Summary

30-month postdoc project at KU Leuven, funded by the Belspo Stereo program.

Forecasting ecosystem response to climate extremes is crucial for understanding and mitigating the impacts of climate change. This project aims to develop data-driven, spatially explicit ecosystem response models using remote sensing data and cGANs.
